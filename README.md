# coffee-order-management 
카페 주문 관리 서비스 

## Client
- 주문 페이지 : React.js    
- 관리자 페이지 : thymeleaf

## Server
- JAVA & SpringBoot
- JUnit5 + TestContainer 

## DB
- MySQL

## 개발 환경
- JDK 17
- SpringBoot 2.6.7
- MySQL 8.0.9

# 기능
## Product
- 판매 상품 리스트 제공
- 관리자 페이지를 통한 제품 등록, 수정, 삭제

## Order
- 주문서 작성 및 저장, 관리 수행
- 관리자 페이지를 통한 주문의 상세정보 확인, 삭제
- 매출 확인
